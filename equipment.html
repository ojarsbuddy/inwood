Bridgemates

We have 60 Bridgemate II terminals and three servers.

The Bridgemate terminals are set to scan a server for sections. Alphabetic section codes used by ACBLscore convert to numeric section codes in the Bridgemates, which lack alphabetic keys. Players enter the correct numeric section code and the correct table number at the beginning of the game to start their Bridgemate terminal session.

System Updates

  • Bridgemate Control Software – version 3.7.6 on 20 Feb 2019
  • Server firmware – version 3.1.1d on 20 Feb 2019
  • Bridgemate firmware – version 3.1.1 on 20 Feb 2019
